Конфиденциальность

ChaosClient не собирает, не хранит и не передаёт никаких личных данных. Без телеметрии, без аналитики, без HWID.

Технологии

  • Fabric Mod Loader для Minecraft 1.21.11
  • Electron для кроссплатформенного лаунчера
  • Java 21 + Mixin
  • Sodium / Lithium оптимизация
